Sheet1

Excel ka basic structure

Har spreadsheet ek grid hai — columns (A, B, C...) aur rows (1, 2, 3...). Jahan column aur row milte hain, wahan banta hai ek cell, jaise B4.

CELL

Cell Reference

Har cell ka apna address hota hai — column pehle, row baad me.

B4 = Column B, Row 4
RANGE

Cell Range

Ek se zyada cells ko colon (:) se jodkar range banti hai.

A1:A5 = A1 se A5 tak
TYPE

Data ke prakar

Cell me number, text, ya formula — teeno aa sakte hain.

42 "Hello" =A1+A2
SHEET

Worksheet vs Workbook

Ek file (workbook) ke andar kai sheets ho sakti hain — jaise iss page ke tabs.

Sheet1, Sheet2, Sheet3...
Sheet2

Formula kaise likhte hain

Har formula = (equal sign) se shuru hota hai. Uske baad aap numbers, cell references, ya functions ko operators se jod sakte hain.

+ − × ÷

Basic Operators

Jod, ghata, guna, bhaag — sabke apne symbol hote hain.

=A1+A2   =A1-A2
=A1*A2   =A1/A2
SUM

=SUM(range)

Ek range ke saare numbers jod deta hai.

=SUM(A1:A10)
AVERAGE

=AVERAGE(range)

Range ka average (ausat) nikalta hai.

=AVERAGE(B1:B6)
MIN / MAX

Sabse chhota / bada

Range me sabse kam ya sabse zyada value dhundhta hai.

=MIN(A1:A10)
=MAX(A1:A10)

💡 Tip: Formula likhne ke baad Enter dabao — result turant dikhega. Practice Sheet5 me try karo.

Sheet3

Thoda advanced — logic wale functions

Yeh functions decision lene aur data dhundhne me madad karte hain — office ke real kaam me sabse zyada use hote hain.

IF

=IF(condition, sahi, galat)

Agar condition sach hai to pehla jawab, warna dusra.

=IF(A1>50,"Pass","Fail")
COUNTIF

=COUNTIF(range, condition)

Range me kitni cells ek condition follow karti hain, gin deta hai.

=COUNTIF(A1:A20,">10")
VLOOKUP

=VLOOKUP(value, table, col, false)

Ek badi table me se ek value ko dhundh kar uske saath wali cell laata hai.

=VLOOKUP("Ravi",A2:C50,2,FALSE)
CONCAT

=CONCATENATE(a,b)

Do ya zyada text values ko jod deta hai.

=CONCATENATE(A1," ",B1)
